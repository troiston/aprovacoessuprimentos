# Learnings Acumulados

Este documento guarda aprendizados persistentes do modo `/orchestrate` entre sessões.
Use junto com `docs/ORCHESTRATOR_METRICS.md`.

## Como usar
- No início da sessão orquestrada: ler este arquivo e aplicar ajustes de execução.
- No fim da sessão: extrair padrões recorrentes a partir das métricas e atualizar este arquivo.
- A cada 5 sessões: consolidar entradas redundantes e manter somente padrões acionáveis.
